# Movie Recommendation Engine

A movie recommendation system developed using Python and machine learning techniques.

## Features

- Recommends movies based on similarity
- Uses movie dataset for recommendations
- Built using Jupyter Notebook

## Files

- `Movie_Recommendation_Engine.ipynb` - Main notebook
- `movies.csv` - Dataset
- `Movie_Recommendation_Engine.html` - Exported notebook
- `PRJ Movie Recommendation.docx` - Project documentation

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## How to Run

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   pip install pandas numpy scikit-learn
   ```

3. Open and run:
   ```bash
   Movie_Recommendation_Engine.ipynb
   ```
